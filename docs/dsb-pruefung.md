# DSB-Prüfung (Datenschutzbeauftragte/r)

Stand: 25.02.2026  
Hinweis: Arbeitsgrundlage, keine Rechtsberatung.

## Ziel
Dokumentieren, ob eine gesetzliche Pflicht zur Benennung eines/einer Datenschutzbeauftragten besteht.

## Prüffragen
1. Werden in der Regel mindestens 20 Personen ständig mit der automatisierten Verarbeitung personenbezogener Daten beschäftigt?
2. Liegen Verarbeitungen vor, die eine Datenschutz-Folgenabschätzung (DSFA) erfordern?
3. Erfolgt eine umfangreiche Verarbeitung besonderer Kategorien personenbezogener Daten?
